A five-book collection

The TypeScript Library

From confident TypeScript to production-ready services, libraries, and monorepos. Five books by Gabriel Anhaia, designed to be read in order or picked individually.

Companion code: github.com/gabrielanhaia/the-typescript-library-examples

Book 1 Book 2 Book 3 Book 4 Book 5

Book 1

TypeScript Essentials

From Working Developer to Confident TS, Across Node, Bun, Deno, and the Browser

Audience: a working developer in any language crossing into TypeScript. Knows programming; needs the language.

Twenty-eight chapters covering the daily-driver TS surface: primitives, narrowing, modules, classes, async, ESLint, Vitest, and the four-runtime daily loop. The entry point of the collection.

Buy on Amazon Companion code

Book 2

The TypeScript Type System

From Generics to DSL-Level Types

Audience: a TS user who wants to wield the type system, not decorate code with it.

Generics, the satisfies operator, mapped and conditional types, infer, template literal types, recursive and branded types, variance, and the type-driven library patterns behind Zod, tRPC, and Hono RPC. Twenty-five chapters.

Buy on Amazon Companion code

Book 3 — Bridge book

Kotlin and Java to TypeScript

A Bridge for JVM Developers

Audience: a working JVM developer (Kotlin-led, Java-aware) crossing into TypeScript.

Static vs structural typing, generics variance, sealed classes vs discriminated unions, threads vs the event loop, Kotlin coroutines vs async/await, Spring/Ktor vs Hono/NestJS/Fastify. Twenty-seven chapters with a Spring-to-Hono worked port.

Buy on Amazon Companion code

Book 4 — Bridge book

PHP to TypeScript

A Bridge for Modern PHP 8+ Developers

Audience: a modern PHP 8+ developer (Laravel, Symfony) crossing into TypeScript.

PHP 8 typed properties and enums mapped to TS, Eloquent and Doctrine mapped to Drizzle and Prisma, FormRequest mapped to Zod, the synchronous-PHP-to-async-TS paradigm shift, the patterns from common Laravel and Symfony codebases ported into TS. Twenty-seven chapters with a Laravel-to-Hono worked port.

Buy on Amazon Companion code

Book 5

TypeScript in Production

Tooling, Build, and Library Authoring Across Runtimes

Audience: a senior TS developer hardening services and libraries. Knows the language; needs production answers.

Every tsconfig flag walked, every build tool compared (tsc, esbuild, swc, tsx, tsup, unbuild, tshy, Vite, Rollup), each runtime first-class, monorepo orchestration with project references, library publishing across npm and JSR, type-level testing, type performance in CI, and migration runbooks. Twenty-eight chapters.

Buy on Amazon Companion code

Reading paths

  • New to TypeScript: Book 1 → Book 2 → Book 5.
  • Working JVM developer: Book 3 substitutes for Book 1 and Book 2; jump to Book 5 for production.
  • Modern PHP 8+ developer: Book 4 substitutes for Book 1 and Book 2; jump to Book 5 for production.
  • Already shipping TypeScript: Book 2 for the type system, Book 5 for production. Skim Book 1.
