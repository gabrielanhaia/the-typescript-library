# The TypeScript Library — landing page

This repo hosts the public landing page for **_The TypeScript Library_**, a
five-book collection by Gabriel Anhaia.

The page is served at:

> [https://gabrielanhaia.github.io/the-typescript-library/](https://gabrielanhaia.github.io/the-typescript-library/)

## What's here

- `index.html` — single-page landing site.
- `style.css` — dark-themed styles matching the cover aesthetic.
- `covers/` — book covers (5 TS Library covers + 8 pocket-guide covers under `covers/other-books/`).

## Companion code

Runnable, CI-verified examples for all five books live in the public examples repo:

> [github.com/gabrielanhaia/the-typescript-library-examples](https://github.com/gabrielanhaia/the-typescript-library-examples)
